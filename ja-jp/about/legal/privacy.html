お客様が当社の製品およびサービスを使用またはアクセすると、お客様は本通知を参照し、その内容を理解したものと見なされます。当社の製品およびサービスとサイトの使用、ならびにプライバシーに関する紛争は、本通知、適用されるサービス利用規約 (損害および紛争解決に関する制限を含む)、およびデータ処理契約、エンドユーザー使用許諾条件、クラウド サービス契約書の対象となります。

当社の発展に伴いビジネスも変化するため、そうした変化を反映するうえで適切と思われる場合には、当社はいつでも本通知を更新できるものとします。本プライバシー通知に重大な変更があった場合には、電子メール、製品内通知、または適用法で規定されているその他の方法で通知します。随時このサイトを参照し、本通知の最新版をご確認ください。

弊社が収集する情報の種類

製品およびサービスを提供するために、弊社は情報を収集しています。この中には、お客様が当社に直接提供するものもあれば、当社の製品およびサービスが自動的に収集するものや、当社が第三者から収集するものもあります。本通知では、単独で、または組み合わせて使用することでお客様個人の特定が可能になるデータを「個人情報」と定義しています。

弊社は、お客様から提供される情報を収集します。 たとえば、お客様が製品およびサービスの購入、アカウントの作成、フォームへの入力、コンテストやプロモーションへの参加、カスタマー サービスの利用、その他、弊社とのやり取りを行う際にお客様の情報を収集します。

お客様が提供する個人データとしては、次のようなものがあります。

  • 連絡先情報 (名前、メール アドレス、住所、電話番号など)。
  • お支払い情報 (決済カードの情報、関連する識別情報、請求先住所、銀行の口座情報など)。
  • アカウントのログイン情報 (当社の一部の製品およびサービスでは、ソーシャル ネットワークの認証情報を使用することがあります)。

この他にも、お客様が購入された製品、興味、人口統計情報、写真、動画、指紋や声紋といった生体認証データ、その他お客様が提供されたデータなどを収集する場合があります。

当社は、お客様による製品およびサービスの利用状況に関する情報、ならびにその製品およびサービスがインストールされているデバイスに関する情報を、収集する場合があります。場合によっては、製品およびサービスがインストールされているデバイスと同じネットワークに接続されている他のデバイスについての情報も自動的に収集します。

たとえば、弊社では、次の情報を収集して使用する場合があります。

  • お客様がご覧になった、あるいは検索された製品や、ご使用になった製品およびサービスに関する情報 (滞在時間などの統計情報も含む)。
  • IP (インターネット プロトコル) アドレス、Cookie 識別子、携帯電話会社、Bluetooth デバイス ID、モバイル デバイス ID、モバイル広告識別子、MAC アドレス、IMEI、広告主 ID、およびお客様がインターネットにアクセスする際にお客様のコンピューターまたはデバイスに自動的に割り当てられるその他のデバイス識別子を含む、お客様のコンピューター、デバイス、アプリケーション、およびネットワークに関する、製品およびサービス、およびその他の Web 分析の方法を通じて収集された情報のほか、ブラウザーの種類と言語、言語設定、バッテリー残量、オン/オフ状態、位置情報、ハードウェアの種類、オペレーティング システム、インターネット サービス プロバイダー、当社の製品およびサービスの使用前後にお客様がアクセスしたページ、アクセス日時、ページごとの滞在時間、製品およびサービス内でお客様がクリックしたリンクおよび閲覧したページに関する情報、さらには環境設定などで製品およびサービスの利用を通じて行われたその他の行為などが含まれます。
  • お客様が当社の製品およびサービスを使用してモバイル デバイスを保護する場合、当社はその製品がインストールされているデバイスの位置情報データを収集することがあります。また、
  • インターネット、アプリ、ネットワークの使用状況に関する詳細 (閲覧した Web サイトの URL やドメイン名、デバイスにインストールされたアプリケーションの情報、トラフィック データなど)、パフォーマンス情報、クラッシュ ログ、他の集計情報または統計情報も収集します。

お客様に当社の製品およびサービスを提供する目的で (マルウェアや迷惑メールの検出および評価ができることなども含めて) 当社は、お客様のファイル (電子メール、添付ファイル、メール アドレス、メタデータ、URL、トラフィック データなどを含む) のデータをスキャン、収集、および保存することがあります。

当社ではこれらの情報を、当社の製品およびサービスと Cookie、Web ビーコンや Web バグ、クリア GIF などの技術を使用して収集します。当社が使用する Cookie などの技術の詳細とお客様の選択肢の詳細については、Cookie に関する通知をご覧ください。

当社では、非個人的な情報ではお客様を特定できないため、非個人的な使用または共有の方法を制限しません。

当社が第三者から収集する情報

当社は、他の情報源からお客様の情報を入手し、当社が直接収集した情報と組み合わせて使用する場合があります。他の情報源から受け取る情報としては、当社で記録修正のために利用する、送付先や支払い情報の更新内容、購入または償還の情報、カスタマー サポートや登録の情報などが該当します。

位置情報

特定の製品およびサービスではお客様の位置情報にアクセスするための許可が必要になります。お客様がこの権限を付与すると、弊社は、GPS、無線通信、Bluetooth 技術を使用してお客様の位置情報を収集します。お客様は、位置情報へのアクセスをモバイル デバイスの設定で制御できます。また、弊社がお客様の IP アドレスを検索し、お客様の位置情報を取得する場合もあります。

収集した情報の使用方法

お客様を保護するため
お客様が当社の製品およびサービスをインストールまたは使用すると、脅威を予測し、お客様、ご使用のデバイス、お客様の情報を保護するため、当社のサービスはお客様のデバイスまたは環境のバックグラウンドで動作します。たとえば、当社は以下の目的で情報を利用することがあります。

  • お客様のデバイスで送受信されたデータを分析して、脅威、脆弱性、ウイルス、不審な活動、スパム、攻撃を識別して阻止し、お客様に潜在的な脅威を通知する。
  • 脅威インテリジェンス ネットワークに参加して調査を実施し、製品とサービスを適応させて新しい脅威に対応する。
  • データを暗号化し、デバイスをロックダウンする。あるいは、データをバックアップまたはリカバリする。
  • 製品およびサービスが正常に実行されていることを確認するため、製品およびサービスの更新状況を確認し、パフォーマンス レポートを作成する。
  • お客様が弊社の識別情報監視製品を使用している場合、お客様のデータの不正利用を検出する。

当社の事業運営の目的で
当社は、収集した情報を以下のような事業目的にも利用します。

  • 個人情報を認証し、生体認証データを使って詐欺を防ぐ。
  • お客様の行動を分析し、当社のサイトおよび製品とサービスの評価、カスタマイズ、改善を行う。これには、新しいセキュリティ技術、データベース、製品やサービスの開発も含まれます。
  • お客様が興味を持つと思われるサプライヤー製品およびサービスを、お客様に通知する。
  • 取引、会計、監査、ライセンス管理、請求、調停、支払、回収活動を行う。
  • カスタマー サポートの提供、問題のトラブルシューティング、契約の管理、ご依頼、質問、コメントへの対応を行う。
  • 特別イベント、プログラム、アンケート、コンテスト、懸賞など各種のプロモーションを促進し、管理する。
  • 市場、トレンド、消費者について、調査、分析を実施する。
  • 当社のブログ、フォーラムをはじめとする公共のコミュニケーションへの投稿を管理する。
  • 潜在的な、または実際の賠償請求、法的責任、禁止行為、および犯罪行為を阻止、検出、識別、調査し、それらに対応する。
  • 法的権利、要件、契約、ポリシーを遵守し、執行する。
  • お客様の同意を得てこれらを行う。

サードパーティの広告

弊社は、お客様が弊社のサイトを閲覧している間もしくは一部の製品およびサービスの利用中に、第三者の広告会社と協力して、お客様に広告を表示または配信しています。このような第三者の広告会社は、広告効果を測定し、広告コンテンツをパーソナライズする目的で、お客様の本サイトでのやり取りに関するデータを収集することがあります。当社と広告パートナーが Cookie などの追跡技術をどう使用しているかについて、またお客様に用意されている選択肢については、当社の「Cookie に関する通知」をご参照ください。下記にご説明したとおり、メールにある配信停止リンクをクリックすることで、第三者がダイレクト マーケティングの目的でお客様の情報を共有することを拒否できます。または、ここをクリックして、当社のマーケティング コミュニケーション ページをご確認ください。

その他の用途
弊社では、ダイレクト マーケティング、個人または市場の調査、詐欺対策、個人データの提供時に開示した用途またはお客様が同意した用途など、正規のビジネス目的で個人データを使用する場合があります。

弊社は通常、製品およびサービスの提供、お客様とのコミュニケーション、弊社の製品およびサービスの宣伝/プロモーション、事業内容の変更または譲渡のため、法的要請、あるいはお客様の同意に基づいて、収集した情報を使用します。

弊社は、以下のように個人情報を共有することがあります。

  • 当社は、お客様を特定しない匿名化された、または集計されたデータを第三者と共有することもあります。

個人情報の保護方法

弊社では、収集および取り扱う個人データを保護するため、管理面および組織面で技術的かつ物理的な安全対策を講じています。弊社のセキュリティ統制は、データの機密性、整合性、適切な水準の可用性を維持するように設計されています。

個人データに関するお客様の選択肢

アカウント
サプライヤーの製品を登録している場合、お客様は製品を通じて、お客様のプロフィールにある個人情報にアクセスしたり、修正したりできます。

サプライヤー製品を登録していない場合でも、当社のいずれかの製品がお客様のデバイスにインストールされているのであれば、その製品をアンインストールすることで、デバイスから サプライヤーが個人情報を収集するのを停止することができます。

当社製品のなかには、当社に送信される個人情報を管理できるものもあります。お客様のアカウントを停止する場合、またはサポートについてご質問がある場合は、お客様のアカウントまたはサポート担当者に連絡するか、製品のサポート ドキュメントを参照してください。

マーケティング コミュニケーション
マーケティング コミュニケーションの受信を停止するには、電子メールに記載されている購読停止のリンクをクリックするか、こちらをクリックして、当社のマーケティング コミュニケーション ページをご確認ください。
